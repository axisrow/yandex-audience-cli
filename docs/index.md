# yac — CLI для Yandex Audience API

`yac` — консольная утилита, **зеркалящая структуру Yandex Audience API «1 в 1»**: версия (`v1`) →
пространство (`management`) → 5 ресурсных групп → 28 эндпоинтов. Команды строятся как
`yac <группа> <операция>`, например `yac segments list` или `yac grants add 777 --user-login user@ya.ru`.

[Yandex Audience](https://audience.yandex.ru/) — сервис Яндекса для работы с сегментами аудиторий:
загрузка списков клиентов (CRM), гео-сегменты, lookalike-аудитории, пиксели, выдача доступов.
`yac` даёт полный доступ ко всем этим возможностям из терминала, без ручного составления
HTTP-запросов.

## Ключевые факты

| | |
|---|---|
| Команда | `yac <группа> <операция> [опции]` |
| Группы | `segments` (16) · `grants` (3) · `pixels` (5) · `accounts` (1) · `delegates` (3) |
| Эндпоинтов | 28, покрытие 100% (машинно гарантировано тестами) |
| API | `https://api-audience.yandex.ru/v1/management` |
| Авторизация | заголовок `Authorization: OAuth <token>` |
| Python | ≥ 3.9 |
| Формат вывода | `json` (по умолчанию) или `table` |

## Быстрый старт

```bash
pip install -e ".[dev]"                    # установка
export YANDEX_AUDIENCE_TOKEN=<ваш OAuth>   # токен (или --token)
yac segments list                          # список сегментов
yac pixels create --name "Мой пиксель"     # создать пиксель
```
