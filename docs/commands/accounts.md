# accounts — аккаунты (1 эндпоинт)

Аккаунты, в которых текущий пользователь является представителем (делегатом).

Путь дан относительно `https://api-audience.yandex.ru/v1/management`.

## Сводка

| Команда | Метод | Путь | Назначение |
|---|---|---|---|
| `list` | GET | `accounts` | Список представительских аккаунтов |

## `list`

```bash
yac accounts list
yac -o table accounts list
```

Команда не принимает аргументов. Пустой ответ (`{"accounts": []}`) — валидный: значит, у текущего
пользователя нет представительств.
