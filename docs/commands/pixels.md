# pixels — пиксели (5 эндпоинтов)

Управление пикселями: создание, изменение, удаление и восстановление (мягкое удаление обратимо).

Пути даны относительно `https://api-audience.yandex.ru/v1/management`.

## Сводка

| Команда | Метод | Путь | Назначение |
|---|---|---|---|
| `list` | GET | `pixels` | Список пикселей |
| `create` | POST | `pixels` | Создать пиксель |
| `update` | PUT | `pixel/{id}` | Изменить пиксель |
| `delete` | DELETE | `pixel/{id}` | Удалить пиксель |
| `undelete` | POST | `pixel/{id}/undelete` | Восстановить удалённый пиксель |

## `list`

```bash
yac pixels list
yac -o table pixels list
```

## `create`

```bash
yac pixels create --name "Мой пиксель"
```

| Опция | Назначение |
|---|---|
| `--name` (обязателен) | имя пикселя |

## `update`

```bash
yac pixels update 724143 --name "Новое имя"
```

| Аргумент/опция | Назначение |
|---|---|
| `PIXEL_ID` (позиционный) | id пикселя |
| `--name` (обязателен) | новое имя |

## `delete`

Мягкое удаление — пиксель можно восстановить (`undelete`).

```bash
yac pixels delete 724143
```

## `undelete`

Восстанавливает недавно удалённый пиксель.

```bash
yac pixels undelete 724143
```

!!! tip "Обратимость"
    В отличие от сегментов, удаление пикселя обратимо: `delete` → `undelete` в пределах срока
    хранения восстанавливает пиксель.
